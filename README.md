# 자동차 경주 게임
## 진행 방법
* 숫자 야구 게임 요구사항을 파악한다.
* 요구사항에 대한 구현을 완료한 후 자신의 github 아이디에 해당하는 브랜치에 Pull Request(이하 PR)를 통해 과제를 제출한다.

## 과제 제출 과정
* [과제 제출 방법](https://github.com/next-step/nextstep-docs/tree/master/precourse)

## 기능 목록
* 난수 클래스 생성
 - 숫자 유효성 테스트
* 자동차 클래스 생성
 - 자동차 이름 유효성 테스트
* 레이싱 카 배열 클래스 생성
 - 자동차들 이름 유효성 테스트
 - 자동차 이름 중복성 테스트
 - ,와 자동차 이름 개수 테스트
* 사용자 입력
* 차량 전진 계산
* 우승자 계산
